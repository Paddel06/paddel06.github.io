Welcome to my little site.
It is inspired by the vita.hacks.guide and also samilops2.gitbook.io with its handy troubleshooting guides, but I have added and adjusted things that I didn't quite agree with on those sites.
This site's main goal is to provide an extensive, but easy to understand resource for all things Vita and hacking it.
Every instruction should make sense and it should be clear what it does, and why, so users better understand how everything works.
Additionally, this site will try to be as operating system and Vita model agnostic as possible, so the experience is the same across the board. Also, images!
Sorry if this site looks like ass, but I suck at webdev. It was created using my fifth grade HTML knowledge and referencing w3schools.com.
Available in English and German (soon).

If you have any questions, you can find me on Discord: Paddel06#9012
